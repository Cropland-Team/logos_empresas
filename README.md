<b>// Logo marcas para a configuração da Cropland //</b>

<b>1 - Gerar o link da imagem no GitHub:</b>

a) Localizar a logo desejada

b) Clicar no link da imagem para abrir

c) Na janela que carregar a imagem, clicar com o botão direito do mouse e escolher "Copiar Link da Imagem"

<img width="390" height="125" alt="image" src="https://github.com/user-attachments/assets/088c1e3e-594b-4e43-a3d0-ed1a04e8d5ad" />



O link copiado será neste padrão: https://github.com/Cropland-Team/logos_empresas/blob/main/Positivo/Bayer.png?raw=true

d) Abrir uma nova janela do navegador, colar o link copiado e dar Enter

e) Se o link estiver correto a imagem será carregada.
O link será alterado para o padrão: https://raw.githubusercontent.com/Cropland-Team/logos_empresas/refs/heads/main/Positivo/Bayer.png

f) Copiar o link definitivo da barra de endereço para utilizar na Cropland.

<b>2 - Configurar na Cropland:</b>
Trocar Empresa > Owner > Editar > Inserir link logo...
